Onboarding of Azure Service Principal has been completed successfully, and you will now want to activate the same in T-Vault and assign permissions to other users who you want to view or rotate password.
To do so please complete the following steps:

Activate Azure Service Principal:

  1. Login to T-Vault.
  2. Navigate to the "Service Accounts" tab.
  3. Select the "Azure Service Principal" from dropdown.
  4. Click the rotate icon to activate Azure Service Principal.

Assigning Permissions:

Once you are finished activating your Azure Service Principal you can assign permissions to other users or groups you want to have permission to view or rotate the password.
  1. Login to T-Vault.
  2. Navigate to the "Service Accounts" tab and select "Azure Service Principal" from dropdown.
  3. Select the Azure Service Principal you want to manage.
  4. Select the "Permissions" button.
  5. Navigate to the Users tab and click on "Add User".
  6. Provide the user name you want to give permissions to and select either read, rotate or deny depending on the level of access you want to give.

Rotate Secret Key:

Once you are finished activating your Azure Service Principal you can rotate the password for the account.
  1. Login to T-Vault.
  2. Navigate to the "Service Accounts" tab and select "Azure Service Principal" from dropdown.
  3. Select the Azure Service Principal you want to manage.
  4. Select "Rotate secret".
